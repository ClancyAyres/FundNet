# FundNet 快速使用指南

## 🎯 应用启动后操作指南

### 1. 应用启动
双击 `FundNet.app` 启动应用后，您会看到一个现代化的界面，包含侧边栏导航和主内容区域。

### 2. 初始界面说明
- **侧边栏**: 包含四个功能模块的导航按钮
- **主内容区**: 显示当前选中模块的内容
- **顶部工具栏**: 显示总资产、总收益等关键信息

### 3. 功能模块使用

#### 📊 基金列表（默认页面）
**作用**: 浏览和搜索基金，添加持仓

**操作步骤**:
1. 在搜索框中输入基金代码或名称
   - 例如: `000001` (华夏成长混合)
   - 例如: `易方达` (搜索包含该关键词的基金)
2. 点击"刷新"按钮获取最新基金数据
3. 找到目标基金后，点击"添加持仓"按钮

**搜索示例**:
- 输入 `000001` 查找华夏成长混合
- 输入 `易方达` 查找易方达系列基金
- 输入 `医疗` 查找医疗相关基金

#### 💼 投资组合
**作用**: 管理您的基金持仓，查看收益统计

**操作步骤**:
1. 点击侧边栏的"投资组合"按钮
2. 在"添加持仓"弹窗中填写:
   - **基金代码**: 您要添加的基金代码
   - **基金名称**: 基金全称
   - **份额**: 您持有的份额数量
   - **成本价**: 您的买入成本价格
   - **分组**: 选择或创建分组（如科技、医疗等）
3. 点击"添加"保存持仓

**示例数据**:
```
基金代码: 000001
基金名称: 华夏成长混合
份额: 1000
成本价: 1.2000
分组: 科技
```

#### 📈 实时监控
**作用**: 查看投资组合的净值走势和收益变化图表

**功能**:
- 净值走势曲线图
- 每日收益柱状图
- 投资组合表现统计
- 可选择时间范围（7天、30天、90天、1年）

#### ⚙️ 设置
**作用**: 配置应用参数和管理数据

**可配置项**:
- 自动刷新间隔（10-300秒）
- 最大历史天数
- 数据源选择
- 显示偏好设置
- 数据备份和恢复

### 4. 常用操作流程

#### 添加第一个持仓
1. 点击"基金列表"
2. 搜索您持有的基金（如输入基金代码）
3. 点击"添加持仓"按钮
4. 填写持仓信息并保存
5. 切换到"投资组合"查看持仓详情

#### 查看收益统计
1. 点击"投资组合"
2. 查看总资产、总收益、当日收益等数据
3. 点击"实时监控"查看图表分析

#### 配置刷新设置
1. 点击"设置"
2. 在"刷新设置"中调整刷新间隔
3. 启用或禁用自动刷新

### 5. 常见问题

#### Q: 搜索不到基金怎么办？
A: 确保网络连接正常，点击"刷新"按钮获取最新基金列表

#### Q: 如何修改已添加的持仓？
A: 在"投资组合"页面找到对应持仓，点击"编辑"按钮

#### Q: 数据丢失了怎么办？
A: 在"设置"页面使用"恢复数据"功能（如果有备份）

#### Q: 如何导出数据？
A: 在"投资组合"或"设置"页面点击"导出数据"按钮

### 6. 示例演示

假设您持有以下基金：

**基金1**: 华夏成长混合 (000001)
- 份额: 1000份
- 成本价: 1.2000元

**基金2**: 易方达消费行业股票 (110022)
- 份额: 500份
- 成本价: 2.5000元

**操作流程**:
1. 在"基金列表"中搜索 `000001` 和 `110022`
2. 分别点击"添加持仓"
3. 填写对应的份额和成本价
4. 在"投资组合"中查看总收益
5. 在"实时监控"中查看净值走势

### 7. 提示和技巧

- **快捷搜索**: 直接输入基金代码搜索更快
- **分组管理**: 按行业或策略创建不同分组
- **定期刷新**: 设置合适的刷新间隔获取最新数据
- **数据备份**: 定期备份重要数据
